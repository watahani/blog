    そもそも Azure AD のトークンのカスタマイズをする理由としては大きく分けると以下の 2 つだと思う。

    1. 連携アプリに対して発行する ID トークン の属性を追加・既存の属性をカスタマイズする。
    2. Azure AD に登録した Web API を利用する用に アクセス トークン をカスタマイズする。

    前者はアプリ側で Azure AD の属性を使いたい場合に利用する。
